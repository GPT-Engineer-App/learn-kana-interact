# learn-kana-interact

A Hiragana and Katakana learning app where user inputs romaji for displayed kana character. After submission, it refocuses on input box and if correct, automatically shows success and advances to next character.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/learn-kana-interact.git
cd learn-kana-interact
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
